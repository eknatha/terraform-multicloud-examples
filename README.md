# Terraform Multicloud Examples

This repository contains Terraform examples for provisioning infrastructure across multiple cloud providers.

## Cloud Platforms

* AWS
* Azure
* Google Cloud
* Openstack

## Repository Structure

aws/ – Terraform examples for AWS
azure/ – Terraform examples for Microsoft Azure
gcp/ – Terraform examples for Google Cloud
openstack/ - Terraform examples for Openstack
modules/ – Reusable Terraform modules

## Examples

* Create EC2 VM on AWS
* Create Virtual Machine on Azure
* Create Compute Engine VM on GCP
* Create storage buckets
* Create VPC networks
* Create VM and resources on Openstack

## Requirements

* Terraform
* Cloud provider credentials configured

## Usage

```
terraform init
terraform plan
terraform apply
```
